Use the editor to explore revisions to this analysis and immediately see the results. Then, you may:
  • Revise this analysis or data, by copying ('forking') the meta-analysis to a new repository, then:
    • Edit or update(1) the data files. Upload files to the editor and create new plots. Upload new data files and plots to your new repository. Then publish in any journal. Please cite openMetaAnalysis in your methods.
  • To revise the methods or display of openMetaAnalysis itself:
    • Edit any of the repository files for openMetaAnalysis to improve the display and method of analysis.
  • Lastly, initiate a 'pull request' for your revisions to be pulled back into the original repository or keep your analysis as a stand-alone.
  • Meta-analyze another topic? Copy ('fork') the example meta-analysis to a new repository. Replace data files with a completely new topic. Upload new data files to editor and create new plots. Upload new data files and plots to your new repository.
  • Click here to see the revision history of openMetaAnalysis.
